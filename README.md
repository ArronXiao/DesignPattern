几种相似的模式对比

### 状态模式 VS 策略模式
*  相同
	两者的类图非常的相似，感觉很像
* 不同
   状态模式不由使用者直接修改和决定，由使用者设置的条件来自行变化
   策略模式则不同，只能由使用者在使用的时候进行抉择
   
   
### 代理模式 VS 装饰器模式
* 相同
   同样是类图很相似
* 不同
  装饰器强调的是增加功能。 代理强调的覆盖 ，重写功能，对原有的类进行限制，控制
